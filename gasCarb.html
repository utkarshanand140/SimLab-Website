Gas Carburising



GasKarb Software

GasKarb is a industrial grade software which is used for simulating the one-dimensional gas carburizing process for the plain carbon and alloy steels. It can also control the process on-line. The code uses finite volume techniques to solve numerically the diffusion equations.


  • Description
  • Additional Information
  • Documentation
Use Cases :-
  • Simulate & Optimize Gas Carburising process.
  • Control Gas Carburising process online.
  • Other heat treatment processes like nitrding.(With certain modifications)
Modelling Features :-
  • Automatic Mesh Generation.
  • 1D Solutions for Co-ordinate systems & Transient Solutions.
  • Supported for various material compositions.
  • Effects of various parameters such as carburising temperature, concentration of carbon, etc can be studied.
Numerical Features :-
  • Implicit Method used for stable results.
  • TDMA used to solve algebric equations.
  • 1D diffusion problem solvable in any co-ordinate.
User Interface :-
  • User friendly interface.
  • Step by Step tutorial guide.
  • Warninig messages in case of diversion from process.
  • Mouse click event for carburisin sampling.


Additional Features :-
  • Fully interactive menu driven software.
  • Online context sensitive help.
  • Colored plots for Graphs & Contours.
  • Provision to import data from other softwares.
  • Experimental data can be inputted in various forms and can be compared with theory on the same plot.
  • All data including Graphs and Contours can be loaded, saved & printed.
  • Fully mouse driven program.
